# Getting Started with ConSync

This guide will help you get up and running with ConSync.

## Prerequisites

Before you begin, ensure you have:

- Java 17 or higher
- Access to a Confluence instance
- API token or PAT for authentication

## Installation

Download the latest release and extract it:

```bash
# Download
curl -LO https://github.com/example/consync/releases/latest/download/consync.jar

# Verify
java -jar consync.jar --version
```
