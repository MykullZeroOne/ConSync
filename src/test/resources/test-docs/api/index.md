# API Reference

Complete reference for ConSync commands and options.

## CLI Commands

- [Sync Command](./sync.md)
- [Validate Command](./validate.md)

## Global Options

| Option | Description |
|--------|-------------|
| `--config, -c` | Path to configuration file |
| `--verbose, -v` | Enable verbose output |
| `--debug` | Enable debug logging |
| `--help` | Show help message |

## Exit Codes

| Code | Meaning |
|------|---------|
| 0 | Success |
| 1 | General error |
| 2 | Configuration error |
| 3 | Authentication error |
| 4 | Sync error |
