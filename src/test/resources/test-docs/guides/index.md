# User Guides

Detailed guides for common ConSync use cases.

## Available Guides

- [Markdown Formatting](./markdown.md) - Supported markdown features
- [Advanced Usage](./advanced.md) - Advanced configuration and usage

## Common Workflows

### Initial Setup

1. Create your documentation structure
2. Configure ConSync
3. Run initial sync

### Daily Workflow

1. Edit markdown files
2. Run `sync --dry-run` to preview
3. Run `sync` to publish

> **Tip**: Use the status command to check sync state!
