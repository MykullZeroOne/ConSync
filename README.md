
# ConSync

ConSync is a command-line tool that synchronizes local Markdown files with Confluence pages. It automates page creation, updates existing content, and ensures version history tracking for seamless documentation management.

## Features
- Convert Markdown to Confluence-compatible content
- Create new pages or update existing ones
- Preserve Confluence version history
- Batch processing of multiple files
- Dry-run mode for safe previews
- Detailed logging and error handling

git clone https://github.com/yourusername/consync.git
cd consync
